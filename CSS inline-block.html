Dispaly:inline

The

display: inline

does not allows to set a width and height on the element.

Dispaly:inline-block

Compared to

display: inline

the major difference is that

display: inline-block

allows to set a width and height on the element.

Display:block

The

display: block

will add a line-break after the element, so the element can sit next line of the elements.

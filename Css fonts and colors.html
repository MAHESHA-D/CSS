Font Varient

The font-variant property specifies whether or not a text should be displayed in a small-caps font.

In a small-caps font, all lowercase letters are converted to uppercase letters.

Font Size

The font-size property sets the size of the text. size 30px

1em is equal to the current font size. The default text size in browsers is 16px. So, the default size of 1em is 16px.(font size 3em)

Use the "vw" unit when sizing the text. 10vw will set the size to 10% of the viewport width.
